
# Tic Tac Toe – Python & Tkinter

Un petit jeu Tic Tac Toe réalisé en Python avec l’interface graphique Tkinter.  
Ce projet est simple mais montre les bases de :
- La programmation événementielle  
- L’utilisation de Tkinter  
- La gestion d’un état de jeu  
- Les interactions utilisateur  

---

## Fonctionnalités

- Interface graphique intuitive en Tkinter  
- Gestion des tours : Joueur **X** puis **0**  
- Détection automatique :
  - des victoires (lignes, colonnes, diagonales)
  - du match nul  
- Message d'état en bas de la fenêtre (joueur courant, victoire, égalité)  
- Réinitialisation automatique après la fin d’une partie  

---

##  Installation

Assurez-vous d’avoir Python installé (3.8+ recommandé).

Aucune dépendance externe : Tkinter est installé par défaut avec Python.
